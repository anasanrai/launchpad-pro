# LaunchPad Pro — Production Readiness Checklist

This document provides a complete guide to deploy LaunchPad Pro to production on Vercel with full Paddle payment integration.

---

## 1. Paddle Sandbox Setup (Testing)

### 1.1 Create Paddle Account & Sandbox Environment

1. Go to [paddle.com](https://paddle.com) and create an account
2. Navigate to **Settings → Sandbox** to enable sandbox mode
3. In Sandbox, create your first product:
   - **Product Name:** LaunchPad Pro - Starter
   - **Price:** $49/month
   - **Billing Cycle:** Monthly
   - **Currency:** USD

4. Repeat for Pro ($99/month) and Agency ($199/month) plans

### 1.2 Get Paddle Sandbox Credentials

1. Go to **Settings → Developer Tools → API Keys**
2. Copy your **Sandbox API Key** (starts with `test_...`)
3. Go to **Settings → Webhooks**
4. Create a new webhook endpoint:
   - **URL:** `https://your-domain.com/api/paddle/webhook`
   - **Events:** Select all subscription-related events:
     - `subscription.created`
     - `subscription.updated`
     - `subscription.cancelled`
     - `subscription.payment_succeeded`
     - `subscription.payment_failed`
5. Copy the **Webhook Secret** (shown once after creation)

### 1.3 Test Paddle Checkout Flow

1. In the Pricing page, click "Upgrade to Pro"
2. Paddle checkout modal should appear
3. Use Paddle test card: **4111 1111 1111 1111** (exp: any future date, CVC: any 3 digits)
4. Complete checkout
5. Verify subscription status updates in your dashboard

## 2. AI Provider Setup

### 2.1 OpenRouter (Recommended - Multi-Model Fallback)

1. Go to [openrouter.ai](https://openrouter.ai)
2. Sign up and navigate to **Keys**
3. Create an API key
4. Copy the key (starts with `sk-or-`)

### 2.2 Google Gemini API

1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Click **Create API Key**
3. Copy the key

### 2.3 OpenAI API (Optional)

1. Go to [openai.com/api](https://openai.com/api)
2. Navigate to **API Keys**
3. Create a new secret key
4. Copy the key (starts with `sk-`)

### 2.4 Anthropic Claude API (Optional)

1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Navigate to **API Keys**
3. Create a new key
4. Copy the key

## 10. Paddle Live Mode Transition

When ready to accept real payments:

### Step 1: Switch to Live API Key
1. In Paddle dashboard, go to **Settings → Developer Tools → API Keys**
2. Copy your **Live API Key** (starts with `live_...`)
3. Update `PADDLE_API_KEY` in Vercel environment variables
4. Redeploy

### Step 2: Update Product IDs
1. In Paddle dashboard, go to **Products**
2. Copy Live product IDs for Starter, Pro, Agency
3. Update `PADDLE_*_PRODUCT_ID` environment variables
4. Redeploy

### Step 3: Update Webhook URL
1. In Paddle dashboard, go to **Settings → Webhooks**
2. Create webhook for production domain
3. Update webhook URL to production domain
4. Copy new webhook secret
5. Update `PADDLE_WEBHOOK_SECRET` in Vercel
6. Redeploy

### Step 4: Test Live Checkout
1. Use real test card or your own card
2. Complete purchase
3. Verify subscription activates
4. Check Paddle dashboard for transaction

## 12. Troubleshooting

### Paddle Webhook Not Firing
- Verify webhook URL is correct and accessible
- Check webhook secret matches in code
- Verify event types are enabled in Paddle dashboard
- Check server logs for errors

### AI Generation Failing
- Verify API keys are correct
- Check rate limits on AI providers
- Verify database connection
- Check error logs for specific API errors

### Subscription Not Activating
- Verify webhook is processing events
- Check database for subscription record
- Verify user ID matches
- Check Paddle dashboard for transaction

### PDF Export Failing
- Verify html2pdf.js is installed
- Check browser console for errors
- Verify Markdown content is valid
- Check available disk space on server
